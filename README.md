# Student Performance Prediction Using Machine Learning

## Project Overview

This project predicts whether a student will pass or fail based on demographic and academic information using Machine Learning techniques.

## Objectives

- Perform data preprocessing
- Build predictive machine learning models
- Compare model performance
- Evaluate accuracy using various metrics
- Visualize important insights

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Machine Learning Algorithms

- Logistic Regression
- Decision Tree
- Random Forest Classifier

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- ROC Curve
- Cross Validation

## Project Files

- Student_Performance_Prediction.ipynb
- StudentsPerformance.csv
- student_performance_model.pkl
- Feature_Importance.csv

## Results

The Random Forest model achieved high prediction accuracy and successfully classified student performance.

## Key Findings

- Academic scores are the strongest predictors of success.
- Random Forest outperformed other models.
- The model generalized well through cross-validation.
